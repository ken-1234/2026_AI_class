# 귀여운 앵무새 갤러리 🦜

사랑앵무와 버크앵무의 귀여운 사진과 일러스트를 감상할 수 있는 웹사이트입니다.

## 기능

- **버크앵무 갤러리**: 버크앵무의 일러스트와 사진을 볼 수 있습니다.
- **사랑앵무 갤러리**: 사랑앵무의 일러스트와 사진을 볼 수 있습니다.
- **탭 기능**: 메인 탭으로 앵무새 종류를 선택하고, 하위 탭으로 일러스트/사진을 선택할 수 있습니다.
- **반응형 디자인**: 모바일, 태블릿, 데스크톱 등 다양한 기기에서 최적화된 경험을 제공합니다.

## 파일 구조

```
ls_AI/
├── index.html      # 메인 HTML 파일
├── styles.css      # 스타일시트
├── script.js       # JavaScript 기능
└── README.md       # 프로젝트 설명
```

## 사용 방법

1. `index.html` 파일을 브라우저에서 엽니다.
2. 상단 탭에서 버크앵무 또는 사랑앵무를 선택합니다.
3. 하위 탭에서 일러스트 또는 사진을 선택하여 갤러리를 감상합니다.

## 커스터마이징

현재 플레이스홀더 이미지가 사용되고 있습니다. 실제 이미지를 추가하려면:

1. `images/` 폴더를 생성합니다.
2. 이미지 파일을 업로드합니다.
3. `index.html`의 `.placeholder-image`를 실제 `<img>` 태그로 교체합니다.

## 기술 스택

- HTML5
- CSS3 (그라데이션, 애니메이션, 반응형 디자인)
- JavaScript (탭 기능, 인터랙션)
- Google Fonts (Noto Sans KR)
